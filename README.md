# VPN Modular Installer - Option B

Dynamic multi-protocol VPN installer with modular scripts and a responsive admin panel.

Run:

```bash
sudo bash main-installer.sh
```

Protocol selection: `1`, `2`, `3`, `1,2`, `1,3`, `2,3`, or Enter/`0` for all.

Panel URL: `http://YOUR_VPS_IP/vpn-panel`
Panel folder: `/var/www/html/panel-admin`

Menu is dynamic based on installed protocols.
