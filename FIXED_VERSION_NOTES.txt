VPN Pro Admin - A-to-Z Fixed + Modern Live UI Version

Added in this build:
1. System Control protocol cards now show mini live install logs inside each card.
2. Mini logs show the latest 5 lines and auto-refresh every 3 seconds.
3. Full Live Install Console remains available below the cards.
4. Dashboard cards auto-refresh every 5 seconds without manual reload.
5. OpenVPN page auto-refreshes users, active devices, logs, status and ports every 5 seconds.
6. OpenConnect page auto-refreshes users, active sessions, logs, status and port every 5 seconds.
7. V2Ray/Xray page auto-refreshes active IP count, port, VLESS link and manual config every 5 seconds.
8. OpenConnect page has a copy button for https://SERVER_IP:PORT.
9. V2Ray/Xray page has copy buttons for VLESS link and manual config.
10. More colorful modern gradient cards, soft cards, toast notifications and live indicators.
11. api_status.php added for background live data updates.
12. install_log.php now supports tail/lines for mini and full live logs.
13. Previous A-to-Z fixes are retained: panel install background jobs, OpenVPN profile port regeneration, firewall persistence, port validation, dependencies, username validation, file permission hardening and no OpenConnect plaintext password display.

Validation performed before packaging:
